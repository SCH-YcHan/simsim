# Tetris MVP

브라우저에서 바로 실행되는 단일 페이지 테트리스입니다. 외부 라이브러리 없이 Vanilla JS/HTML/CSS만 사용합니다.

## 실행 방법
1. 이 폴더에서 `index.html`을 브라우저로 열면 바로 실행됩니다.

## 조작키
- ← / → : 좌우 이동
- ↓ : 소프트 드랍(누르고 있는 동안 빠르게 낙하)
- ↑ : 시계 방향 회전
- Space : 하드 드랍(즉시 바닥까지)
- P : 일시정지/재개
- R : 재시작

## 규칙 요약
- 보드 크기: 10 x 20
- 테트로미노: I, O, T, S, Z, J, L
- 랜덤 생성: 단순 랜덤
- 라인 삭제: 한 줄 이상 꽉 차면 삭제 후 위 줄 내려옴
- 점수: 1/2/3/4줄 삭제 시 100/300/500/800점
- 게임 오버: 새 블록 스폰 위치가 막혀 있으면 종료
